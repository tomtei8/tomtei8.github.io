##  Local Development

To run this site locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/tomtei8/tomtei8.github.io.git
   ```

2. Navigate to the project folder:
   ```bash
   cd tomtei8.github.io
   ```

3. Open index.html in your browser:
   ```bash
    # On macOS
    open index.html

    # On Windows
    start index.html

    # On Linux
    xdg-open index.html
    ```

4. Or use a local development server (Python):
    ```bash
    # Python 3
    python -m http.server 8000

    # Python 2
    python -m SimpleHTTPServer 8000
    ```

Then visit http://localhost:8000

##  Customization

Want to use this template for your own resume? Feel free to fork and customize!

1. Update the personal information in index.html

2. Modify colors in style.css (search for gradient colors #667eea and #764ba2)

3. Add your own projects or sections

4. Replace placeholder links with your actual profiles
